Calculator Test Strategy

Visual

  • All required buttons appear with correct label (0-9; +; -; x; รท; =; C; '.'')
  • Calculator display appears
  • The display starts with a 0

Button Functionality

  • All number buttons display to the screen when clicked (0-9) and the clear button returns the display to 0

Basic Calculations - Single Integer

  • Addition (1 + 2 = 3)
  • Subtraction (3 - 1 = 2)
  • Division (6 / 3 = 2)
  • Multiplication (4 * 2 = 8)

Basic Calculations - Larger Integers

  • Addition (12 + 23 = 35)
  • Subtraction (36 - 15 = 21)
  • Division (999 / 9 = 111
  • Multiplication 10 x 99 = 990)

Basic Calculations - Decimals

  • Addition (1.2 + 2.3 = 3.5)
  • Subtraction (3.6 - 1.5 = 2.1)
  • Division (99.9 / 2 = 49.95
  • Multiplication (10.2 x 99.2 = 1011.84)

Chaining Equations

  • Addition (1 + 2 + 3 = 6 + 4 + 5 = 15)'
  • Subtraction (100 - 1 - 9 = 90 - 20 - 5 = 65)
  • Multiplication (2 x 5 x 3 = 30 x 3 x 2 = 180)
  • Division (100 / 2 / 5 = 10 / 1 = 10 / 2 = 5)
  • Mixed Equation (100 / 2 * 5 = 250 - 200 = 50 + 5 = 55)

Clear Button Functionality

  • Clears Addition Chain Midway (10 + 5 + 2; C; 2 + 2 = 4)
  • Clears Subtraction Chain Midway (10 - 2 - 1; C; 4 - 1 = 3)
  • Clears Multiplication Chain Midway (5 x 2 x 3; C; 3 x 4 = 12)
  • Clears Division Chain Midway (100 / 2 / 2; C; 20 / 5 = 4)
